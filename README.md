# cinema_flask
Репозиторий, содержащий приложение об учете фильмов, созданное на Flask

**Приложение позволяет:**
- Просматривать полную информацию о фильмах
- Фильтровать и сортировать фильмы
- Искать нужные фильмы
- Добавлять отзывы (только для авторизованных пользователей)
- Добавлять, изменять и удалять фильмы (только для авторизованных пользователей)

**Особенности:**
- База данных создана на SQLlite
- При регистрации, пароль хэшируется и недоступен для просмотра, так как его нельзя расшифровать
- Удобная главная страница. Каждый постер содержит название, оригинальное название, рейтинг IMDB и возрастное ограничение. При нажатии на карточку показывает полную информацию о фильме
- Фильтрация происходит по возрасту и жанру
- Сортировка происходит по названию (А-Я и Я-А), рейтингу (По возрастанию и по убыванию), а также по году выхода (По возрастанию и по убыванию)
- Быстрый поиск фильмов по названию
- Четкое разделение контроллеров, моделей и представлений (MVC)
- Полный набор тестов на pytest для проверки функционала. Тесты проходят в базе данных SQLite in-memory, которая повторяет основную базу данных, но никак не связана с нею и удаляется после окончания тестирования

**Структура проекта:**
- controllers/ (Обработка запросов и взаимодействие между моделью и представлением)
- models/ (Данные и логика приложения)
- views/ (Визуальное отображение данных)
- static/ (Дизайн элементов для views)
- /fonts/ (Шрифты)
- /icons/ (Иконка для заголовка страницы)
- /images/ (Постеры фильмов)
- /js/ (Логика для models и controllers)

**Установка**
1. Клонируйте репозиторий
- `git clone https://github.com/VladosssE/cinema_flask.git`
- `cd cinema_flask`

2. Установите зависимости
- `pip install -r requirements.txt`

3. Запустите приложение
- `python app.py`
- При первом запуске программы app.py автоматически создаст базу данных и пользователя.

4. Перейдите по ссылке (http://127.0.0.1:5000)

**Тестирование**
1. Перейдите в корень проекта
2. Запустите тесты
- `pytest -v`
Тесты используют SQLite in-memory, поэтому реальная база данных не затрагивается.

**Зависимости**
- Flask
- Flask-SQLAlchemy
- Flask-Login
- PyMySQL
- pytest

**Примечания**
- Для постеров и изображений используется папка static/images
